# Introduction 
Cascading Style Sheets 层叠样式表 — 表现Html文件样式
***
## Font-size — 字体大小
```<p style="font-size:......">......</p>```</br>
*单位：像素（px）、百分比（%）、em值*</br>
字体在浏览器默认显示大小为**16px**=**100%**，以此类推
## Color — 颜色
```<p style="color:......">......</p>```</br>
*表示方式：RGB、16进制编码、颜色名称*</br>
color:red ▏color:rgb(225,10,10) 参数值可用数字也可用百分值 ▏color:63B1DC
